# qpubsubserver
Publish-Subscribe Server using Qt

* Prototyping with Qt 5.8
* Using MQTT protocol
  * https://github.com/emqtt/qmqtt
  * http://mqtt.org/
* Using Google Protocol Buffers
  * https://github.com/google/protobuf
